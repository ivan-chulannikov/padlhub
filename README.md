# Расписание первых пробных тренировок

Тестовое задание: расписание бесплатных первых тренировок Падл хаб, представленное в формате турнирного расписания.

## Реализовано

- Недельная навигация по датам.
- Фильтрация по станции.
- Группировка тренировок по дням.
- Карточки со временем, длительностью, тренером, станцией и свободными местами.
- Окно с подробной информацией о тренировке.
- Состояние без результатов и сброс фильтров.
- Адаптивная версия для desktop, tablet и mobile.
- Семантическая разметка и базовая клавиатурная доступность.

Запись и оплата намеренно не реализованы в соответствии с условием задания.

## Стек

- React 19
- TypeScript
- Vite
- CSS Modules

## Запуск

```bash
npm install
npm run dev
```

Production-сборка:

```bash
npm run build
```

## Принятые решения

Данные вынесены в отдельный типизированный модуль и сейчас представлены моками. Это позволяет показать интерфейс без зависимости от закрытого API и при необходимости заменить источник данных на REST API без переработки UI.

Визуальное направление опирается на контрастный фирменный стиль Падл хаб и структуру турнирного расписания. Основной сценарий пользователя — быстро выбрать дату и станцию, сравнить доступные занятия и открыть подробности.
